## 🏃💨 Getting started

1. Clone this repo with ``git clone https://github.com/calebesg/nlw-copa.git``
2. Move yourself to the nlw-copa directory: `cd nlw-copa`
3. Run the ``npm install`` add all dependencies
4. Run ``npm start`` start app
